Sales Data Analysis Project

This project analyzes sales data to derive insights on sales performance by region and other key metrics.

Project Description

This project involves cleaning sales data, visualizing total sales by region, and using interactive filters to analyze sales performance. The goal is to identify trends and make data-driven decisions based on the insights gained from the analysis.
